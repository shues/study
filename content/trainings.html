HTML

HTML - это не язык программирования. Расшифровка этой аббревиатуры выглядит так HyperText Markup Language. А перевод этой фразы звучит как «язык гипертекстовой разметки».

Чтобы увидеть полный список тем по курсу языка разметки веб страниц, переходите на страницу "Изучение HTML".

CSS3

CSS - читается как Си Эс Эс. Это язык при помощи которого веб разработчики описывают, как должны выглядеть элементы страницы. Точно так же как и в случае с HTML, CSS это лишь аббревиатура, расшифровка ее на английском языке Cascading Style Sheets, а по русски это будет "каскадные таблицы стилей".

Почему именно таблицы, что за стили они описывают, как они стали каскадными и главное - как их использовать вы узнаете на курсе, список тем которого вы можете посмотреть на странице "Изучение CSS"

JavaScript

JavaScript - обычно произносят как джава скрипт. Настоящее же ег о название ECMAScript. И как ни странно он не имеет ничего общего с популярным языком программирования Java (не считая того что и тот и другой являются серьезными языками программирования).

JavaScript оживляет веб приложения, делая их максимально интерактивными, и существенно расширяет возможности разработчика.

На странице "Изучение JavaScript" вы найдете полный список тем, которые можно изучить на наших курсах.
